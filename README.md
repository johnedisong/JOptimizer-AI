# JOptimizer-AI

Sistema inteligente de análisis de código Java que utiliza machine learning para clasificar e identificar oportunidades de optimización en código Java. Construido con Python y scikit-learn, esta herramienta analiza métricas de código para determinar si los segmentos de código son óptimos o subóptimos.

## Características Principales
- Modelos avanzados de machine learning (Random Forest y Árboles de Decisión)
- Análisis automatizado de métricas de código
- Entrenamiento de modelos con datos históricos
- Sistema de predicción fácil de usar para nuevo código
- Métricas de evaluación completas

## Stack Tecnológico
- Python 3.x
- scikit-learn
- pandas
- numpy
- joblib
